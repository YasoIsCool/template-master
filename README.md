## 项目启动

```js
npm run cnpm    // 使用cnpm下载全部依赖,并且跳过已安装的node_modules

npm run serve    // 运行全部项目

npm run build     // 打包全部项目

```

## 移除项目

```
npm install rimraf -g
npm run rm-node // 移除全部的 node_modules
```

注意：如果下载报错,可进入目录分开运行。
